# Text Search
文本搜索可以划分为以下三个层次：
1. 字符串搜索：将文本当作不具有语义的数据进行搜索
2. 词义搜索：在搜索时考虑文本中词的词义
3. 语义搜索：在搜索时考虑文本的整体语义

Question answering 也可看作是文本搜索的智能形态。
